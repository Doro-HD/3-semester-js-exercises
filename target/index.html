Up 
get /birds/{birdFamily}
(birdsBirdFamilyGet)
returns all birds by their bird family

Path parameters

birdFamily (required)
Path Parameter —

Return type

array[birds]

Example data

Content-Type: application/json
[ {
  "scientificName" : "scientificName",
  "name" : "name",
  "id" : "046b6c7f-0b8a-43b9-b35d-6489e6daee91",
  "birdFamily" : "birdFamily",
  "wingSpand" : 0.8008281904610115
}, {
  "scientificName" : "scientificName",
  "name" : "name",
  "id" : "046b6c7f-0b8a-43b9-b35d-6489e6daee91",
  "birdFamily" : "birdFamily",
  "wingSpand" : 0.8008281904610115
} ]

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

found bird family

404

No bird family with matching name
